# Social Network Globe Analyzer

Aplicação de análise de redes sociais com visualização 3D do globo terrestre usando algoritmos de detecção de comunidades.

## Instalação

```bash
npm install
```

## Executar

```bash
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000)

## Build para Produção

```bash
npm run build
npm start
```
